# BAML Port: Track A (prompt-only + logprobs) for gpt-oss-20b

## Overview

Port the two Track A inference scripts to use BAML for prompt management and structured output,
targeting `openai/gpt-oss-20b` served via vLLM's OpenAI-compatible API.

## File Structure

```
baml_src/
  generators.baml     # generates baml_client/
  clients.baml        # GptOss20b openai-generic client
  track_a.baml        # PerturbEffect enum + PredictPerturbEffect function
baml_client/          # generated by baml-cli generate (gitignored)
baml/
  track_a_prompt_only.py
  track_a_logprobs.py
```

## BAML Definitions

**Client (`clients.baml`):** `openai-generic` provider reading `VLLM_API_BASE`, `VLLM_API_KEY`,
`VLLM_MODEL` from env. Static options: `temperature 1.0`, `top_p 1.0`. Per-call overrides
(seed, reasoning_effort, max_tokens) injected via `ClientRegistry`.

**Types (`track_a.baml`):**
- `PerturbEffect` enum: `Up | Down | NoEffect`
- `PredictPerturbEffect(pert: string, gene: string) -> PerturbEffect`
- Zero-shot prompt matching `mlgenx._PROMPT_ZERO` + `ctx.output_format`

## Script: track_a_prompt_only.py

- For each (row, seed) pair: build `ClientRegistry` with seed + reasoning_effort,
  call `await b.PredictPerturbEffect(pert, gene, baml_options=BamlCallOptions(client_registry=cr))`
- Enum → (pred_up, pred_down): Up=(1.0, 0.0), Down=(0.0, 1.0), NoEffect=(0.0, 0.0)
- Average across seeds 42/43/44; same cache/CSV/zip output as original
- No regex parsing — BAML handles structured extraction

## Script: track_a_logprobs.py

- Reads same env vars; creates `openai.AsyncOpenAI(base_url=..., api_key=...)`
- Prompt: zero-shot template + `<answer>X</answer>` tag instruction (Python-formatted)
- Call: `client.chat.completions.create(..., logprobs=True, top_logprobs=N, seed=seed)`
- `--no-reasoning` passes `chat_template_kwargs: {enable_thinking: false}` via `extra_body`
- Same softmax logprob extraction (`prediction_from_logprobs`) + caching + CSV/zip as original

## Configuration

Both scripts read from env vars (with defaults):
- `VLLM_API_BASE` — default `http://localhost:8000/v1`
- `VLLM_API_KEY` — default `token-abc123`
- `VLLM_MODEL` — default `openai/gpt-oss-20b`

## Model Notes

`gpt-oss-20b` is a reasoning model (like 120b). `reasoning_effort` applies.
Default `max_tokens` for prompt-only: 65536. Logprobs default: 8192.
